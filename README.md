## 🔎 GitFind

Aplicação web desenvolvida com **React** para buscar usuários e repositórios do GitHub de forma simples, rápida e intuitiva.

---

## 📌 Sobre o projeto

O **GitFind** é uma aplicação que consome a API pública do GitHub para exibir informações detalhadas de usuários e seus repositórios.

O objetivo do projeto é praticar conceitos de **consumo de APIs**, **manipulação de estado** e **componentização com React**, aplicando tudo em um cenário real.

---

## 🚀 Tecnologias utilizadas

* React
* JavaScript
* HTML5
* CSS3
* Fetch API

---

## 🌐 Integração com API

A aplicação utiliza a API pública do GitHub para obter dados em tempo real:

* 🔗 Dados do usuário
* 🔗 Lista de repositórios

Exemplos de endpoints utilizados:

```bash
https://api.github.com/users/{username}
https://api.github.com/users/{username}/repos
```

---

## 🎯 Funcionalidades

* 🔍 Buscar usuários do GitHub pelo username
* 👤 Exibir foto, nome, username e bio
* 📁 Listar repositórios do usuário
* 🔄 Consumo de dados em tempo real via API
* 🎨 Interface moderna e responsiva


## 📦 Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/gitfind.git
```

Acesse a pasta:

```bash
cd gitfind
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```
