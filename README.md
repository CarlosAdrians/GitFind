**# 🔎 GitFind

Aplicação web desenvolvida com **React** para buscar perfis e repositórios do GitHub de forma simples e intuitiva.

---

## 📌 Sobre o projeto

O **GitFind** permite pesquisar usuários do GitHub e visualizar suas principais informações, como:

* Foto de perfil
* Nome e username
* Bio
* Lista de repositórios

Tudo isso em uma interface moderna e responsiva.

---

## 🚀 Tecnologias utilizadas

* React
* JavaScript
* HTML5
* CSS3

---

## 🎯 Funcionalidades

* 🔍 Buscar usuários do GitHub pelo username
* 👤 Exibir informações do perfil
* 📁 Listar repositórios do usuário
* 🎨 Interface limpa e amigável

## 📦 Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/gitfind.git
```

Acesse a pasta do projeto:

```bash
cd gitfind
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm start
```
